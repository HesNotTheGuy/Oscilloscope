# Screenshot Capture Guide

Save each screenshot as the filename listed below into this folder (`docs/screenshots/`).
Use the in-app screenshot button (camera icon) or Windows Snip & Sketch (Win+Shift+S).
Recommended resolution: 1280×800 or higher.

---
